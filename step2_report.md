## Exercise Routine
- Name  
- Description (optional)  
- List of routine items:
  - Rest time (between exercises)  
  - Exercise  
  - Rest time between sets  
  - Amount of sets  
  - Weight (optional)  
  - Can have either:
    - Reps  
    - Duration
